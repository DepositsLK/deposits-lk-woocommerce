Deposits.lk allows you to automate your Bank Deposit Slips.

== Description ==
Deposits.lk allows you to automate your bank deposits quickly and easily. It integrate with your current woocommerce store and acts as a new payment option. Your customers can select it the payment option and they can upload their payment slip. 

The plugin mainly uses Deposits.lk Service (https://deposits.lk/) to do the transactions. The plugin shares the Order information such as Customer Details (First Name, Last Name, E-mail ) and Order details (Currency, Ordered Items, ID). 

== Installation ==
1. Upload plugin zip file or Install directly from WordPress plugin directory
2. Activate the plugin
3. Navigate to Woocommerce > Settings > Payments
4. Click Deposits.lk and Fill API Key and API Secret with the values provided to you by the developer
5. Click Save Changes
